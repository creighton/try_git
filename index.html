Experience API

The Experience API is a service that allows for statements of experience to be delivered to and stored securely in a Learning Record Store (LRS).

Spec on GitHub >

The ADL Learning Record Store is an example implementation of the Learning Record Store defined in the Experience API Specification. This system is a Django web app that is designed to be installed on an Ubuntu server. All of the source is openly licensed under the Apache 2.0 License and freely available on GitHub.

LRS on GitHub >

There is also a hosted ADL LRS based on the GitHub project. This is a complete working instance of that project and allows for users to sign up and use the LRS. The data in the system is not permanent. Since this is only for demonstration purposes and testing, the LRS will be updated and data wiped periodically.
